Your private wedding command center — no logins, no cloud, no subscriptions.

◆ Getting Started ◆
1

Open your planner file in a browser

Double-click the .html file that matches your tier — wedding-hq-lite.html, wedding-hq-standard.html, or wedding-hq-premium.html. It opens directly in Chrome, Safari, Firefox, or Edge. No install needed.

2

Your data stays on your device

Everything you type is saved automatically to your browser's localStorage — private by design. No account, no server, no one else can see it. As long as you use the same browser on the same device, your data persists indefinitely.

3

Back up your data regularly

Open the Settings panel (gear icon, top-right) and click Export JSON to download a backup file. Store it in Google Drive, Dropbox, or email it to yourself. To restore, use Import JSON and select your backup file.

4

Works offline, forever

Once opened, Wedding HQ Studio works with no internet connection. Your purchase is a one-time download — no subscription, no expiry date, no forced updates.

5

Print any section

Use your browser's File → Print (or Ctrl/Cmd + P) from any section. The layout switches to a clean, ink-friendly print view automatically. Use "Save as PDF" to create shareable copies.

◆ FAQ ◆
Can I use this on multiple devices?

Yes — just copy the .html file to any device and open it. To transfer your data between devices, export a JSON backup on one device and import it on the other via Settings.

What if I clear my browser history or cache?

Clearing cookies/localStorage will erase your saved data. Always keep a JSON backup (Settings → Export JSON) so you can restore it instantly.

Can I share this file with my partner?

Yes — your purchase covers personal household use. Share the file with your partner or co-planner on the same device or copy it to their device with a JSON export of your current data.

How do I upgrade to a higher tier?

Visit our Etsy shop and purchase the tier you'd like. Then export your current data as JSON, open the new tier file, and import your backup — your data carries over seamlessly.

Which browser works best?

Chrome and Edge give the best experience. Safari and Firefox work great too. Avoid opening the file from inside a ZIP archive without extracting first — always extract to a regular folder before opening.

I see a blank screen / nothing loads.

Make sure you extracted the ZIP and are opening the file from your computer (not directly from inside the ZIP). If it still doesn't load, right-click the file → Open With → Chrome (or your preferred browser).
