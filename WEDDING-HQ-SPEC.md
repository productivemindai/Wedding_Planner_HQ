# DELIVERABLE 1 — PRODUCT POSITIONING

**Product:** WEDDING HQ — Hyperlinked Digital Wedding Planner
**Built exclusively for:** the modern DIY bride who plans on her iPad.

**Core promise:**
Your private, editorial-style luxury wedding command center — 160 hyperlinked pages that feel like a high-end physical binder but work instantly in GoodNotes/Notability. Zero generic clip-art. Zero overwhelm.

**Differentiation:**
- Persistent top tabs + home/back icons on every page
- Section-level TOCs + master clickable navigation map
- Neutral-luxury aesthetic (taupe/cream/olive) that photographs beautifully
- Every high-utility tool pre-built and fillable:
  - Vendor scorecard
  - Master budget with payment log
  - Full wedding-day run sheet
  - Seating planner
- Conversion-focused Etsy packaging: instant-download PDF + bonus editable Canva file + printables

# DELIVERABLE 5 — DESIGN SYSTEM

## Color Palette

| Token | Name | Hex | Use |
|---|---|---|---|
| `--color-paper` | Cream | `#F8F5F0` | Page background |
| `--color-surface` | Warm Beige | `#EDE6DB` | Cards, secondary bg, TOC headers |
| `--color-accent` | Muted Olive | `#A8B5A0` | Active tabs, buttons, olive highlights |
| `--color-text` | Deep Taupe | `#3A2F2A` | All body text |
| `--color-header` | Warm Brown | `#5C4638` | Section/page headers |
| `--color-border` | Soft Sand | `#D9D0C4` | Table borders, dividers, card outlines |
| `--color-btn-text` | Cream (on olive) | `#F8F5F0` | Button text on olive background |

## Typography

### Font Pair (Google Fonts — free)

| Role | Font | Weight | Size | Notes |
|---|---|---|---|---|
| Cover / Section Dividers | Playfair Display | 600 italic | 42pt | Hero titles |
| Page Titles | Playfair Display | 600 | 28pt | H2 |
| Card / Section Sub-heads | Playfair Display | 400 | 20pt | H3 |
| Tab Labels | Inter | 700 | 14pt | All-caps |
| Body / Form Labels | Inter | 400 | 13pt | Standard body |
| Captions / Small Labels | Inter | 400 | 11pt | Muted color |

### Canva Font Equivalents
- Playfair Display → available in Canva
- Inter → available in Canva

## Spacing System

| Token | Value | Use |
|---|---|---|
| `--margin-page` | 60px | All page outer margins |
| `--pad-block` | 40px | Content block vertical padding |
| `--gap-element` | 24px | Standard gap between elements |
| `--line-height-body` | 1.55 | Body text |
| `--line-height-header` | 1.20 | Headers |

## Component Specifications

### Top Tab Bar
- Height: 50px
- Border radius: 20px
- Drop shadow: 2pt soft
- Passive: `--color-surface` bg + `--color-text` label
- Active: `--color-accent` fill + `#F8F5F0` text
- Number of tabs: 12
- Min tap target: 44px wide × 50px tall

### Section Divider Pages
- Background: full-bleed cream (`#F8F5F0`)
- Large section number: 120pt Playfair Display, 8% opacity taupe
- Section title: 42pt Playfair italic, `--color-header`
- Horizontal taupe rule above + below title: 1pt `--color-border`
- "Begin Section →" pill CTA: olive fill, cream text

### Checkboxes
- Size: 20 × 20px, rounded square
- Border: 1.5pt `--color-border`
- Radius: 4px
- Checked: `--color-accent` fill + white checkmark

### Form Fields / Fill-in Lines
- Underline style: 1pt `--color-border`
- Boxed style (worksheets): full border 4px radius, 36px height, transparent bg
- Label: Inter 11pt all-caps muted above field

### Tables
- Header row: `--color-surface` fill, 700 weight, 11pt caps
- Alternating rows: `#F8F5F0` / white
- Cell border: 1pt `--color-border`
- Cell padding: 12px horizontal, row min-height 36px (finger-friendly)

### Vendor / Stat Cards
- Background: `#FFFFFF`
- Border: 1pt `--color-border`
- Radius: 12px
- Accent left stripe (optional): 3pt olive

### Icons
- Style: 24px minimal line, 1.5pt stroke, rounded caps
- Color: `--color-accent` (#A8B5A0)
- Source: Lucide Icons (free, open source)
- Key set: home, arrow-left, arrow-right, check, dollar-sign, calendar, user, heart, star, clipboard, phone, mail, map-pin
